# Decide — Frontend

This repository hosts the frontend for the Decide micro-decision assistant.

The backend runs privately on the owner's machine and is not included here.

## Files

- `login.html` — Sign in page
- `signup.html` — Create account page  
- `app.html` — Main application
- `config.js` — API endpoint configuration (update when tunnel URL changes)
- `index.html` — Redirects to login
